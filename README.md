## About The Project

![Product Name Screen Shot][product-screenshot]

Inside the expanse of autonomous driving, a critical point is the navigation algorithm, which takes decisions based on sensors data.
This project presents a simulated environment where a machine learning algorithm will learn how to drive the Sailboat based on wind and position data.
